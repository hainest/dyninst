{

enum BPatch_edgeType { CondJumpTaken, CondJumpNottaken, UncondJump, NonJump }}



{

BPatch_edgeType getType()}



{

Return a type describing this edge. A CondJumpTaken edge is found after a conditional branch, along the edge that is

taken when the condition is true. A CondJumpNottaken edge follows the path when the condition is not taken.

UncondJump is used along an edge that flows out of an unconditional branch that is always taken. NonJump is an edge

that flows out of a basic block that does not end in a jump, but falls through into the next basic block.}
